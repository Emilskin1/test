permissions calculator

the link above is the permissions calculator please click on it, when you have the permissions calculator open you need to go back to the bot application tab, click on general information, and copy your client id, now go back to the permissions calculator and paste your bots id in the box in the lower left corner, click on all the boxes to make sure your bot has access to everything, next click on the link in the lower left corner of the permissions tab, now select the server you want your bot in, make sure all the boxes are blue with a white checkmark then click on authorize, your bot is now added in your server!

now you need to program the bot to go online, click on your file explorer, then desktop, make a folder called discordbot, now close the file explorer and click on the search bar in the lower left or upper right corner next to your windows icon and type cmd, in the command prompt you type cd desktop, then cd discordbot, then npm init, then click on enter 9 times, then type yes, now type npm install node.js, then npm install discord.js, now type node .

now that we're in visual studio code hover your mouse over you discordbot folder, click on new file, name this file index.js, open index.js, now type this in your index.js folder: const Discord = require('discord.js');
